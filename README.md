### Instalation
- Install GCC `sudo apt-get install gcc`
- Install FPC `sudo apt-get install ...`
- Make a symlink of FPC `sudo ln -s /usr/lib/fpc/2.4.4/ppc386 /usr/local/bin/fpc`
