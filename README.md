##  AcmMD

### Description

### Requirements
- PHP 5.1.0+
- curl enabled
- FPC
- GCC

### Interface languages
- Russian
- Romanian

##  Instalation
### FPC
`apt-get install fpc`<br>
Try to run `fpc` and if it doesn't work run:<br>
`sudo ln -s /usr/lib/fpc/2.6.0/ppc386 /usr/local/bin/FPC`
### GCC
`sudo apt-get install gcc`

### SSH2 PHP extesion
`apt-get install libssh2-php`

### Mod_Rewrite
a2enmod rewrite
/etc/init.d/apache2 restart
